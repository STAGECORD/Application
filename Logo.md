# StageCordpro Logo Design Specifications

## Logo Overview
Text-based logo for STAGECORD PRO

---

## Design Specifications

### Logo Text
- **Text Content**: "StageCordpro" or "STAGECORD PRO"
- **Position**: Absolute
- **Position Coordinates**: Left: 143px, Top: 12px
- **Dimensions**:
  - Width: 319px
  - Height: 50px

### Typography
- **Font Family**: Outfit
- **Font Style**: Normal
- **Font Weight**: 700 (Bold)
- **Font Size**: 40px
- **Line Height**: 50px
- **Text Transform**: Uppercase

### Colors
- **Primary Color**: #FFFFFF (White)
- **Background**: Typically displayed on dark backgrounds

---

## Placement Guidelines

### Top Navigation/Header
- Position at top left of main content area
- Left margin: 143px from viewport edge
- Top margin: 12px from top
- Provides clear brand identity above main content

### Usage Context
- Main header logo across all pages
- Positioned to the right of the sidebar (which is 126px wide)
- Creates consistent branding throughout the application

---

## Variations

### Standard Logo
- Full text: "STAGECORD PRO"
- Used on desktop and tablet views
- Full 319px width

### Mobile Logo (if needed)
- Potential abbreviated version: "SC PRO"
- Responsive sizing for smaller screens

---

## Brand Guidelines

### Do's
- Always use Outfit font family
- Maintain uppercase text transform
- Use on dark backgrounds for optimal contrast
- Keep proper spacing around logo (minimum 12px padding)

### Don'ts
- Don't change font weight from 700
- Don't use colors other than white on dark backgrounds
- Don't compress or stretch the logo
- Don't place on light backgrounds without adjustment

---

## Technical Implementation

### HTML Structure
```html
<div class="logo">
    <h1>STAGECORD PRO</h1>
</div>
```

### CSS Requirements
- Position: absolute or fixed depending on scroll behavior
- Font: Outfit (Google Fonts import required)
- Z-index: High value for visibility above content
